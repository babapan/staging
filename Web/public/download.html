Pertanyaan Umum

Jawaban untuk pertanyaan yang sering ditanyakan.

Apakah aman install APK dari luar Play Store?
Aman, selama kamu mengunduh dari sumber resmi yaitu web.chatmeapp.my.id. APK kami dibuild langsung oleh developer dan diserve via HTTPS. Hindari mengunduh APK KyuLive dari situs lain karena bisa dimodifikasi.
Kenapa belum ada di Play Store?
Kami sedang dalam proses review Play Store. Sementara itu, distribusi via APK langsung lebih cepat untuk update dan tidak terbatasi kebijakan store.
Bagaimana cara update ke versi baru?
Saat ada versi baru, aplikasi akan menampilkan notifikasi untuk download update. Atau kamu bisa kembali ke halaman ini dan download versi terbaru — install di atas versi lama tanpa perlu uninstall (data tidak hilang).
Bisa install di iOS / iPhone?
Belum. Saat ini KyuLive hanya tersedia untuk Android. Versi iOS sedang dalam roadmap pengembangan.
Ada masalah saat install. Harus hubungi siapa?
Email ke support@chatmeapp.my.id dengan info: model HP, versi Android, dan screenshot error. Tim akan respon dalam 1×24 jam.
