Kebijakan Privasi

Terakhir diperbarui: 27 April 2026

Privasi Anda penting bagi kami. Dokumen ini menjelaskan data apa yang kami kumpulkan, bagaimana kami menggunakannya, dan hak Anda atas data pribadi sebagai pengguna KyuLive.

1. Data yang Kami Kumpulkan

Saat Anda menggunakan KyuLive, kami mengumpulkan data berikut:

  • Data akun: username, email, password (terenkripsi), tanggal lahir, foto profil.
  • Data perangkat: model HP, sistem operasi, versi aplikasi (untuk debugging).
  • Data interaksi: pesan chat, postingan, komentar, hadiah yang dikirim/diterima.
  • Data teknis: alamat IP, log aktivitas, timestamp login (untuk keamanan).
  • Data transaksi: riwayat pembelian kredit (kami tidak menyimpan data kartu kredit).

2. Cara Kami Menggunakan Data

Kami menggunakan data Anda untuk:

  • Menyediakan dan memelihara fitur aplikasi (chat, gift, party room, leaderboard, dll).
  • Mengamankan akun dari penyalahgunaan, spam, atau hack.
  • Mengirim notifikasi penting (verifikasi email, reset password, peringatan keamanan).
  • Meningkatkan kualitas layanan berdasarkan analitik anonim.
  • Memenuhi kewajiban hukum jika diminta oleh pihak berwenang.

3. Penyimpanan Pesan Chat

Pesan chat disimpan di server kami selama akun aktif untuk mendukung fitur riwayat, offline messages, dan moderasi. Pesan yang dilaporkan dapat ditinjau oleh tim moderasi sesuai keperluan investigasi.

4. Berbagi Data dengan Pihak Ketiga

Kami tidak menjual data pribadi Anda. Data hanya kami bagikan dengan:

  • Penyedia infrastruktur: AWS (hosting), Brevo (email), ImageKit (CDN gambar).
  • Otoritas hukum: jika ada permintaan resmi dengan dasar hukum yang sah.

5. Keamanan Data

Kami menerapkan langkah keamanan industri standar:

  • Password disimpan dengan hashing bcrypt (tidak pernah disimpan dalam bentuk plain text).
  • Semua koneksi menggunakan HTTPS/TLS untuk enkripsi data dalam transit.
  • Database hosted di provider tepercaya dengan kontrol akses ketat.
  • Audit log untuk aktivitas administratif.

6. Hak Anda atas Data Pribadi

Sesuai UU PDP (Pelindungan Data Pribadi) Indonesia, Anda berhak untuk:

  • Mengakses data pribadi yang kami simpan tentang Anda.
  • Memperbarui data yang tidak akurat melalui pengaturan akun.
  • Menghapus akun beserta data yang terkait (kecuali yang diwajibkan disimpan oleh hukum).
  • Mengajukan keberatan atas penggunaan data tertentu.

Untuk mengajukan permintaan, kirim email ke support@chatmeapp.my.id dengan subjek "Permintaan Data Pribadi". Kami akan merespon dalam 30 hari kerja.

7. Cookie & Local Storage

Aplikasi mobile menggunakan local storage untuk menyimpan token login dan pengaturan pengguna. Versi web menggunakan cookie httpOnly untuk session authentication. Tidak ada tracking cookie pihak ketiga.

8. Anak di Bawah Umur

KyuLive tidak ditujukan untuk anak di bawah 13 tahun. Jika kami mengetahui ada akun anak di bawah umur, akun tersebut akan dihapus dan datanya dimusnahkan.

9. Penyimpanan Internasional

Data Anda disimpan di server AWS region Asia Pacific (Singapore atau Jakarta). Tidak ada transfer data ke negara dengan standar privasi yang lebih rendah.

10. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini. Perubahan signifikan akan diumumkan melalui aplikasi dan/atau email. Tanggal "Terakhir diperbarui" di atas selalu menunjukkan versi terkini.

11. Kontak

Untuk pertanyaan tentang privasi atau permintaan data, hubungi kami:
